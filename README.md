# GObject Editor Tool

**GObject Editor Tool** is an in-game addon tool designed to make working with game objects easier, cleaner, and more visual. It provides a compact control interface for selecting, moving, rotating, adjusting, and managing objects without relying only on typed commands.

Made for builders, world editors, server owners, and anyone who wants a smoother object-editing workflow.

---

## ✨ Features

- Clean in-game object editing interface
- Easy access to object movement controls
- Rotation and positioning controls
- Scale and adjustment controls
- Undo support
- Search window for locating game objects
- Popout controls window
- Compact layout designed for active building and editing
- Built for a cleaner workflow than command-only editing

---

## 📦 Installation

1. Download or clone this repository.
2. Place the addon folder into your WoW client AddOns directory:

```text
World of Warcraft/Interface/AddOns/
```

3. Make sure the folder name stays as:

```text
GObjectEditor
```

4. Launch the game.
5. At the character select screen, click **AddOns**.
6. Make sure **GObject Editor Tool** is enabled.
7. Enter the game and use the tool.

---

## 🧭 Basic Usage

Once the addon is enabled, open the GObject Editor Tool in game.

Use the interface to control and edit selected game objects through the available buttons and controls.

The tool is intended to help with:

- Moving objects
- Rotating objects
- Adjusting object placement
- Searching for game objects
- Managing edits through a cleaner visual workflow

---

## 🔍 Search Window

The search button opens a dedicated search window.

From there, you can enter search text and run a search based on what you typed. This keeps searching separate from the main control panel and avoids cluttering the main interface.

---

## 🪟 Popout Controls

The tool includes a popout controls window so editing controls can be separated from the main window when needed.

This is useful when you want quick access to controls while keeping the main interface out of the way.

---

## ↩️ Undo Support

Undo support is included to help recover from recent object-editing actions.

Use this carefully while building, especially when placing or adjusting multiple objects.

---

## 🛠️ Recommended Use

This addon is best used by players or server staff who already have access to game object editing commands or server-side tools.

It is meant to improve the workflow around object editing, not replace the required server permissions or backend command support.

---

## ⚠️ Notes

- This tool is intended for supported server environments where game object editing commands are available.
- Required permissions depend on your server setup.
- Some functions may require specific server-side command access.
- Always test object edits in a safe area before using the tool in finished zones.
